# HEARTBEAT — Proactive Checklist

> SPESION reads this file during each heartbeat cycle (default: every 30 minutes).
> Each item is a proactive check. Mark as ✅ when completed for the current cycle.

## Every Heartbeat (30 min)

- [ ] Check calendar for upcoming events in the next 2 hours
- [ ] Review any pending tasks that are overdue
- [ ] Monitor portfolio for significant price movements (>3% change)
- [ ] Check for new messages that need follow-up
- [ ] Verify system health (Ollama, APIs, disk space)

## Every 2 Hours

- [ ] Scan ArXiv for new papers in priority categories
- [ ] Check news for topics in the user's interest areas
- [ ] Review and consolidate recent memories
- [ ] Update energy/mood estimates based on time of day

## Daily (Morning — 08:30)

- [ ] Generate Morning Brief (calendar + tasks + fitness + portfolio)
- [ ] Sync investment data (IBKR + Bitget → Notion)
- [ ] Review yesterday's journal entry (if exists) for patterns
- [ ] Check for birthdays/important dates in contacts

## Daily (Evening — 22:00)

- [ ] Prompt for evening journal if not done yet
- [ ] Summarize day's interactions into daily memory log
- [ ] Run reflection cycle (extract learnings from today)
- [ ] Archive completed tasks

## Weekly (Sunday — 10:00)

- [ ] Generate Weekly Review (goals progress, patterns, insights)
- [ ] Run deep memory compaction (merge redundant memories)
- [ ] Update opinion confidence scores based on new evidence
- [ ] Generate 3 strategic ideas based on accumulated knowledge
- [ ] Check for workspace file updates needed (SOUL.md, etc.)

## Active Hours

- Start: 07:00
- End: 23:30
- Timezone: Europe/Madrid
- During sleep hours: Only critical alerts (security, major market events)
